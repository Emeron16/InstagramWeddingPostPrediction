\f0\b\fs36 \cf0 Description\
\pard\pardeftab720\sa240\partightenfactor0

\f1\b0\fs24 \cf0 This Flask-based web application predicts the popularity category ("Low", "Medium", "High") of social media posts based on structured data, captions, and hashtags. It also suggests the best future dates for posting based on predicted popularity.\
The app uses a pre-trained multi-input deep learning model that processes structured data, captions, and hashtags to provide popularity predictions.\
